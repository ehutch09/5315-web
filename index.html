What is Docker

Docker is an open source software virtualization container platform that was initially released on March 13th 2013. Docker utilize containers to deploy, run, and manage applications for developers allowing for flexibility and portability since the container format keeps the application and code the same no matter what environment it is deployed in. Docker uses Linux and Windows as its operating systems and utilizes Linux kernel functionality to isolate resources and applications creating lightweight images. This container approach allows for the elimination of set-up time and compatibility configuration for developers as the application code is organized and ready to use. The ability to share, test, and deploy work collaboratively is the big benefit of using Docker container software.

What is Nginx

NGINX is a free, open source, HTTP web server that is used to power several high profile websites used by many around the world. This server was created by Igor Sysoev in 2004 and runs on Unix, Linux and many other operating systems. NGINX uses event driven architecture to handle requests efficiently, and has been shown handle four times as many requests at Apache. This web server has a simple configuration and low resource consumption that makes hosting web pages easy.
